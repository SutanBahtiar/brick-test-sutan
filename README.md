## About The Project
Web scraping project for getting top products.

### Stack
1. Java 1.8
2. Spring Boot
3. Maven
4. [Apache Http Client](https://hc.apache.org/httpcomponents-client-5.1.x/)


## How to start
```sh
make start or mvn spring-boot:run 
```
